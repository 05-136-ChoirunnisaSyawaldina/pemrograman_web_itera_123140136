## Tugas Praktikum: Sistem Manajemen Perpustakaan Sederhana

Ini adalah proyek untuk memenuhi tugas praktikum mata kuliah Pemrograman Berorientasi Objek. Program ini adalah simulasi sistem manajemen perpustakaan sederhana yang ditulis dalam bahasa Python dan menerapkan konsep-konsep inti OOP.

## Penjelasan Singkat dan Fitur

Program ini mengelola koleksi item di perpustakaan. Item dapat berupa Buku atau Majalah. Sistem ini memungkinkan admin (pengguna) untuk:

1. Menambahkan Item: Menambahkan item baru (Buku atau Majalah) ke dalam koleksi perpustakaan.
2. Melihat Daftar Item: Menampilkan semua item yang saat ini berstatus "Tersedia".
3. Mencari Item: Mencari item spesifik berdasarkan ID Item atau Judul (pencarian judul bersifat case-insensitive).
4. Peminjaman & Pengembalian: Mengubah status item dari "Tersedia" menjadi "Dipinjam" (check out) dan sebaliknya (check in).
